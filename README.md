然后分别尝试使用了两层128个神经元作为隐藏中间层 dropout为0.3
			使用了两层128个神经元作为隐藏中间层 dropout为0.4
			一二层64，128个神经元作为隐藏中间层 dropout为0.3

分别都迭代了300次
